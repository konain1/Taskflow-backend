# Taskflow Backend

Backend setup for the Taskflow assignment.

## Prerequisites

- Node.js

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/konain1/Taskflow-backend.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Taskflow-backend
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
